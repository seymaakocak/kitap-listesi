##  Özellikler

- Kitap ekleme
- Kitapları listeleme

-  Kitap silme

## Çalıştırma Adımları

### 1. Repoyu klonla
```bash
git clone https://github.com/seymaakocak/kitap-listesi.git
cd kitap-listesi
```

### 2. Bağımlılıkları yükle
```bash
npm install
```

### 3. Geliştirme sunucusunu başlat
```bash
npm run dev
```

Tarayıcıda `http://localhost:5173` adresini aç.

### 4. Production build al
```bash
npm run build
```
